# Controle de escola
Sistema para controle de escola.

## Softwares necessários

* Editor de código-fonte
* Node.js
* Git

## Instalação

```bash
# Clone o repositório
$ git clone https://github.com/imsantosrodrigo/controle-escola.git

# Acesse o diretório
$ cd controle-escola

# Instale as dependências
$ npm install
```

## Executando o sistema

```bash
# Inicie o servidor
$ npm start
```

## Tecnologias

* HTML

* Node.js
